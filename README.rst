Installation
############

以下のコマンドを入力してください。

.. code-block:: bash

    $ cd ~/
    $ git clone https://github.com/u-akihiro/my-vimrc.git
    $ cd ~/my-vimrc
    $ bash symlink.sh

What's NeoBundle?
#################

NeoBundleとは、vimプラグインの管理ツールです。以下のものがインストールされます。

+-----------------------+-----------------------------------------------------+
| ツール                | 説明                                                |
+=======================+=====================================================+
| ``neocomplcache``     | 入力補完                                            |
+-----------------------+-----------------------------------------------------+
| ``neosnippet``        | スニペット補完                                      |
+-----------------------+-----------------------------------------------------+
| ``jedi-vim``          | Pythonのスニペット                                  |
+-----------------------+-----------------------------------------------------+
| ``riv.vim``           | reStructuredText (REST) 用プラグイン                |
+-----------------------+-----------------------------------------------------+
